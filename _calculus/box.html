A box is a set of the form $I=[a_1, b_1]\times \dots \times[a_n, b_n]\subseteq\mathbb{R}^n$ where $[a_k, b_k]$ is a closed interval for $k\in\{1, 2, \dots, n\}$.
The volume of a box $I$ is $\mu(I)=\prod_{k=1}^n(b_k-a_k)$.
