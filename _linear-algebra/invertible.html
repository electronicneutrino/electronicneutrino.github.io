A linear transformation $T:U\to U$ is invertible if there exists a linear transformation $T^{-1}:U\to U$ such that $TT^{-1}=T^{-1}T=I$.
Furthermore if $T:V\to V$ is left or right invertible, then $T$ is invertible.

Concepts

As usual a function is invertible if and only if its bijective.
